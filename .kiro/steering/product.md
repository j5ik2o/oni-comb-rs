# Product Overview

oni-comb-rs (鬼昆布) は Rust 向けの LL(k) パーサーコンビネータライブラリである。関数型プログラミングの原則（Functor, Monad, Pure）に基づき、小さなパーサーを組み合わせて複雑な文法を宣言的に記述できる。

## Core Capabilities

- **汎用パーサーコンビネータ基盤**: 任意の入力型 `I` に対して動作するジェネリックな `Parser<'a, I, A>` を提供
- **演算子によるパーサー合成**: `+`(連接), `|`(選択), `*`(左スキップ), `-`(右スキップ) で直感的な文法定義
- **豊富なビルトインパーサー**: 要素マッチング、正規表現、繰り返し、先読み、バックトラック等
- **ドメイン別パーサークレート**: URI, HOCON, Cron 式など実用的なパーサーを同梱

## Target Use Cases

- 構造化テキスト（設定ファイル、プロトコル、DSL）の解析
- RFC 準拠のパーサー実装（URI 等）
- 教育・学習用途でのパーサーコンビネータの理解

## Value Proposition

- Rust の型システムとライフタイムを活かした安全なパーサー構築
- `Rc` ベースのクロージャ共有により、パーサーの再利用とクローンが容易
- crates.io で公開済み、プロダクション利用可能なライブラリ
